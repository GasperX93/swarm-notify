Encrypted notifications and messaging on Swarm — no servers, no cloud, no accounts.

1

Publish Identity

Publishes your public keys and overlay to a Swarm feed at topic keccak256("swarm-identity-" + ethAddress). Plaintext — public by design. This is how others discover your encryption keys.
2

Discover Bob

Reads Bob's identity feed to get his public key. You need this for ECDH key exchange — it lets you derive a shared encryption secret without ever transmitting a key.
3

Send Message to Bob

ECDH derives a shared secret from Alice's private key + Bob's public key. AES-256-GCM encrypts the message. The encrypted blob is uploaded to Swarm and the mailbox feed is updated. Only Alice and Bob can derive this key — neither party ever sends it.
5

Notify Bob (On-Chain)

ECIES-encrypts {sender, overlay, feedTopic} with Bob's public key. Emits a Notification event on Gnosis Chain. Only needed for first contact — subsequent messages are found via feeds. Cost: ~22,000 gas (~0.00002 xDAI).
7

Check Inbox

Reads Bob's mailbox feed (Bob→Alice), derives the shared secret, and decrypts. This shows the reply from the other side.

Bob

ETH: —
PubKey: —
1

Publish Identity

Same operation for Bob. Both identities are now discoverable by ETH address on Swarm. Anyone can read the feed — privacy comes from encryption, not access control.
2

Discover Alice

Same in reverse. Both now have each other's public keys — ready for encrypted communication. No key exchange server needed.
4

Read Messages from Alice

Reads Alice's mailbox feed from Swarm. Derives the same shared secret using Bob's private key + Alice's public key (ECDH is symmetric). Decrypts with AES-256-GCM. Neither party ever sent the key!
6

Poll Notifications

Queries Gnosis Chain for Notification events matching keccak256(bobAddress). ECIES-decrypts each with Bob's private key. Failed decryptions (spam or not for you) are silently discarded. This is how you discover new contacts — no server needed.
7

Reply to Alice

Bob sends an encrypted reply. Same ECDH + AES-256-GCM flow, but on Bob's own mailbox feed (Bob→Alice). Two feeds per conversation — each owned by the sender.
